Privacy Policy — ClarityRead

Last updated: 2025-10-23

Who we are

ClarityRead is created and maintained by an independent developer. This extension is offered to help make reading on the web easier via dyslexia-friendly fonts, reflowed reading, read-aloud, and a local summarizer.

Summary — no remote data transfer by default

ClarityRead does not send page content or user selections to any remote server by default. The summarizer and text-to-speech use local browser capabilities. All user data described below is stored locally in the browser unless you explicitly enable an optional cloud feature (which will prompt you and request your consent).

What we store locally

  • Preferences (font choice, reflow, high contrast, voice settings) — stored in chrome.storage.local.
  • Saved reads / summaries — stored locally and accessible only to the extension in your browser profile.
  • Reading statistics — aggregated locally for charts and usage insights.
  • Export / Import — you may export saved reads & settings as a JSON file and re-import them at your discretion. Exported files are under your control.

What we do NOT collect

  • We do not collect or store browsing history, page URLs, or page content on our servers by default.
  • We do not collect personal data or identifiable information unless you explicitly provide it (for example, in a support email).

Optional cloud features (explicit opt-in only)

If an optional remote/cloud feature exists or is added later (for example, server-side summarization or backup), it will be clearly marked in the UI and will only be used after obtaining your explicit consent. If you enable such a feature, the extension will tell you exactly what data will be sent and to which host. You can revoke the feature at any time from extension settings.

Text-to-speech

ClarityRead uses your browser's built-in speech synthesis (the Web Speech API) — voices available are provided by your browser/OS. No audio or text is sent to external services for TTS unless you explicitly enable an optional cloud TTS feature (which will prompt you first).

Security & retention

Data stored locally remains on your device and will remain until you delete it or uninstall the extension. If you enable any optional cloud feature, that remote data retention will be described in the opt-in dialog. We encourage you not to upload extremely sensitive personal information to any service.
