Support & Contact — ClarityRead

If you have a bug, feature request, or need help, please use one of the channels below.

Email

For direct support, contact me at ttonnaagburu@gmail.com. Please include your browser (Chrome/Edge), extension version, and a short description of the issue.

Report an issue (recommended)

If you use GitHub, open an issue on the repository so I can track and respond:

Open GitHub Issue

Replace YOUR_GITHUB_USERNAME with your GitHub username or link to the correct repo. If you don't want to use GitHub issues, email is fine.

Quick troubleshooting

  • Ensure the extension is enabled in chrome://extensions.
  • Try disabling other reader or accessibility extensions — they may conflict.
  • If read-aloud fails, check your browser's speech voices and permissions.

Feedback & donations

If you'd like to support development, consider starring the GitHub repo or sharing the extension with friends. Donations/Patreon links can be added later.

Local logs (optional)

To help debug issues you can copy the extension console logs and paste them into an issue or email. Avoid pasting full-page content — include only relevant error lines and a short reproduction steps list.
